Fitur Lengkap

Semua yang kamu butuhkan untuk mengelola supermarket

📦

CRUD Lengkap

Kelola data barang, kategori, transaksi, dan pengguna dengan operasi Create, Read, Update, Delete yang mudah.

🔗

Relasi Database (JOIN)

Query menggunakan JOIN antar tabel barang, kategori, dan transaksi untuk tampilan data yang lengkap dan akurat.

📋

Laporan & Report

Laporan transaksi per periode, rekap bulanan, top barang terjual, dan fitur cetak langsung dari browser.

🔍

Search & Filter

Pencarian real-time di semua halaman — barang, transaksi, kategori, hingga pengguna dengan filter fleksibel.

🔐

Login & Registrasi

Sistem autentikasi lengkap dengan session, hashing password bcrypt, registrasi akun baru, dan logout aman.

🏗️

OOP PHP

Kode terstruktur menggunakan konsep Object-Oriented Programming dengan class Database, Auth, Barang, Transaksi, dan User.

Cara Install

Siap dijalankan dalam beberapa langkah mudah

1️⃣

Download

Clone atau download repository dari GitHub ke komputer kamu.

2️⃣

Import Database

Import file supermarket.sql ke phpMyAdmin atau MySQL.

3️⃣

Konfigurasi

Sesuaikan koneksi database di file config/database.php.

4️⃣

Jalankan

Buka browser dan akses localhost/supermarket. Selesai!
